API Setup
=========

You can install the library from `Pypi`_ :

.. code-block:: bash

    python3 -m pip install -U FixRaidenBoss2

:raw-html:`<br />`

For windows, use the command below:

.. code-block:: bash

    py -3 -m pip install -U FixRaidenBoss2
