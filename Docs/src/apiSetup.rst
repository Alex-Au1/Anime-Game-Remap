API Setup
=========

You can install the library from `Pypi`_ :

.. code-block:: bash

    python3 -m pip install -U AnimeGameRemap

:raw-html:`<br />`

For windows, use the command below:

.. code-block:: bash

    py -3 -m pip install -U AnimeGameRemap

.. _Pypi: https://pypi.org/project/AnimeGameRemap/
